# Input System

## Input Multiplexer (priority order)

1. ResultOverlay processor (blocks all if result showing)
2. InfoBar processor (back button, challenge list expand)
3. ActionBar processor (seed selection, reset)
4. SwipeDetector (gravity control in play area)

## Controls

- **Swipe:** Change gravity direction
- **Double-tap:** Reset seed position + gravity to DOWN
- **Tap seed button:** Switch active seed type
- **Tap reset:** Clear all plants, reset nutrients, clear zen progress (per-level)
- **Tap InfoBar challenge count:** Expand/collapse challenge list
