# Technical Stack

- **Framework:** LibGDX 1.12+
- **Physics:** Box2D (bundled with LibGDX)
- **Language:** Kotlin
- **Platform:** Android
- **Audio:** Procedural WAV generation (no asset files)
- **Rendering:** ShapeRenderer + custom GLSL shaders
- **UI:** Custom-drawn (no Scene2D)
- **Persistence:** LibGDX Preferences + JSON files
- **Build:** Gradle with Android plugin
